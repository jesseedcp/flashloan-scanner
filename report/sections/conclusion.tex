\section{Conclusion}

This report studies flash loans as a key DeFi primitive and examines both their normal applications and their security relevance. We first explain the interaction process between flash loan providers and users, then design a rule-based framework for identifying flash loan transactions, and finally prepare for an empirical analysis on Ethereum transaction data.

The completed version of this report will combine protocol understanding, implementation details, transaction analysis, and case studies to provide a structured view of flash loan behavior in the DeFi ecosystem.
