# 5566project

This repository is organized as a workspace with multiple parallel parts:

- `flashloan-scanner/`: Go backend scanner and APIs
- `frontend/`: frontend application
- `docs/`: project notes and implementation plans
- `report/`: report sources
- `translated/`: translated paper sources

The backend code that originally belonged to the `flashloan-scanner` project is kept inside the `flashloan-scanner/` subdirectory.
